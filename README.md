## Why Use Resistance Hook?

Unlike the original version, which was limited to a simple debug console, Resistance Hook (Ultimate) transforms *The Saboteur* with a high-performance DirectX 9 UI and a set of built-in mods that work out of the box.

- Cinematic “Noir” Mod Menu (NUMPAD 7): A custom-built, hardware-accelerated interface designed to match the game’s aesthetic. Includes smooth animations, high-contrast visuals, and intuitive navigation.
- Integrated Memory Cheats: No external tools required. Includes toggleable features such as:
  - Infinite Health & Ammo

- Hybrid Navigation System: Combines full-screen sections with expandable lists for sub-options, all within a single seamless interface.
- Classic Lua Console (F8): The original console remains available for advanced scripting and debugging.

## Installation
1. Download the `d3d9.dll` from the latest release.  
2. Extract it into your main Saboteur directory (where `Saboteur.exe` is located).  
3. Make sure the `/mods` folder is present to enable scripts, spawners, and cheats.  
4. Launch the game and press NUMPAD 7.
